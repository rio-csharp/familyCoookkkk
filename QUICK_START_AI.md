## 📝 还需要你做的事情（3个步骤）

### 步骤 1️⃣：申请腾讯混元 API（10分钟）

1. 访问：https://cloud.tencent.com/product/hunyuan
2. 点击 **立即使用**，注册或登录腾讯云
3. 进入 **腾讯混元控制台**：https://console.cloud.tencent.com/hunyuan
4. 开通服务（有免费额度给新用户）
5. 创建 API 密钥：
   - 左侧菜单 → **访问管理** → **API 密钥管理**
   - 点击 **新建密钥**
   - **复制并保存** SecretID 和 SecretKey（✅ 这很重要！）

### 步骤 2️⃣：配置环境变量（5分钟）

1. 打开 **微信开发者工具**
2. 左侧菜单 → **云函数** → 右键点击 `api`
3. 选择 **云函数配置**
4. 在 **环境变量** 中添加：

```
变量名: HUNYUAN_SECRET_ID
值: 你的SecretID

变量名: HUNYUAN_SECRET_KEY  
值: 你的SecretKey
```

5. 点击 **保存**

### 步骤 3️⃣：更新前端 UI（15分钟）

## 💡 功能说明

### AI 食谱优化
- **作用**：改进用户输入的食谱
- **优化内容**：
  - 使描述更吸引人和有食欲
  - 使步骤更清晰和易操作
  - 补充烹饪技巧
- **调用时机**：用户点击"AI优化"按钮

### AI 菜谱推荐
- **作用**：基于已有食材推荐菜谱
- **用法**：用户输入冰箱里的食材，AI 推荐搭配菜谱
- **应用场景**：
  - 解决"今天吃什么"的问题
  - 充分利用手里的食材
  - 减少食材浪费

### AI 问答（已集成，可选实现）
- **作用**：回答用户的烹饪问题
- **示例问题**：
  - "番茄炒蛋怎样不出水？"
  - "如何让蛋更蓬松？"
  - "番茄什么时候加盐？"

## ❓ 常见问题

**Q: API 不工作怎么办？**
A: 
1. 检查环境变量是否正确（无多余空格）
2. 查看云函数日志：开发者工具 → 云函数 → api → 日志
3. 确认腾讯混元服务已开通

**Q: 费用会不会很贵？**
A: 非常便宜。新用户有免费额度（通常 2-3 个月的使用量），之后按 Token 计费，1000个Token 约 0.01 元。

**Q: 可以离线使用吗？**
A: 可以。如果想完全离线，可以后续部署 Ollama（详见主文档）。

**Q: 会不会泄露用户数据？**
A: 数据只发送到腾讯云，由腾讯通过合规存储。不会存储用户的具体食谱内容。
