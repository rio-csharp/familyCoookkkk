## API 接口列表

### 新增接口
1. `getHotRecipes` - 获取今日热门食谱
   - 返回：`{ hotRecipes: [...] }`

2. `uploadAvatar` - 上传用户头像
   - 参数：`{ fileID: "云存储fileID" }`
   - 返回：`{ avatar, user }`

### 修改接口
1. `updateProfile` - 更新用户资料
   - 新增参数：`avatar`（可选）
   - 参数：`{ nickname, phone, avatar }`

2. `getRecipeDetail` - 获取食谱详情
   - 自动增加浏览量 `views++`

## 使用说明

### 用户头像管理
1. 进入"我的"页面
2. 点击"编辑资料"
3. 点击头像选择或更换按钮
4. 从相册/相机选择图片
5. 点击"保存"完成

### 社区热度轮播图
1. 进入"社区"页面
2. 顶部显示"今日热门"轮播图
3. 展示当天点赞和浏览最高的5个食谱
4. 点击任意卡片进入食谱详情
5. 轮播图自动播放，也可手动滑动

## 技术要点

1. **云存储上传**：使用 `wx.cloud.uploadFile` 上传图片
2. **热度算法**：`hotScore = likes * 3 + views`
3. **日期筛选**：只统计当天创建的食谱
4. **轮播组件**：使用微信原生 `<swiper>` 组件
5. **响应式设计**：适配不同屏幕尺寸
6. **渐变背景**：使用 CSS 线性渐变
7. **数据持久化**：修改后立即保存到数据库
